# Solo Choir — score-roll player

Public deploy of the take-home QR score-roll page (GitHub Pages).
Source lives in the private main repo at `web/roll/`; this repo mirrors the self-contained player only.

Open with a score in the URL fragment, e.g. `https://yvwHY.github.io/solo-choir-roll/#<payload>` (the studio app generates the QR/link).
